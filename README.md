# About This Repos

## How to install

### Clone This Repos
```
git clone https://github.com/bewith/atcoder.git
```

### srcフォルダ以下を削除
```
rm -rf src
```

### VSCODEのRemoteContainerで接続
- 左下の緑色をクリック
- Open Containerを選択

## How to use
@see https://github.com/kyuridenamida/atcoder-tools

```
atcoder-tools gen abs
cd src/abs/PracticeA/
atcoder-tools test
```
